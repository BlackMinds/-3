# 万界仙途 - 项目进度文档

## 一、已完成功能

### 1.1 账号系统 ✅
- 注册/登录,JWT 鉴权,水墨风登录页

### 1.2 角色创建 ✅
- 五行灵根选择(金/木/水/火/土),道号,头像上传

### 1.3 等级系统 ✅ (新)
- **上限 200 级**,打怪获得独立等级经验
- 经验曲线按段递增(1-30快/30-80中/80-150慢/150-200很慢)
- 属性加成按段递增:
  - Lv1-50: 每级 气血+5/攻击+2/防御+1/身法+1
  - Lv51-100: 气血+10/攻击+4/防御+2/身法+2
  - Lv101-150: 气血+20/攻击+8/防御+4/身法+3
  - Lv151-200: 气血+40/攻击+15/防御+8/身法+5
- 角色信息条双进度条(修为绿+等级金)

### 1.4 境界系统 ✅
- 8大境界:练气→筑基→金丹→元婴→化神→渡劫→大乘→飞升
- 修为积满自动突破,管解锁地图/功法/内容

### 1.5 地图历练系统 ✅
- **25 张地图**(T1~T10)
- T9: 太虚圣境/鸿蒙秘境/万界战场
- T10: 天道试炼/永恒之巅
- 每个地图 3 种普通怪 + 1 Boss
- **多怪波次**: 每次遭遇随机 1-5 只怪,逐个打,血量延续

### 1.6 战斗系统 ✅
- 回合制自动战斗,最多 50 回合
- **10 种 debuff**: 灼烧/中毒/流血/冻结/眩晕/减速/脆弱/攻击削弱/束缚/封印
- **8 种 buff**: 攻击提升/防御提升/速度提升/暴击提升/护盾/持续回血/伤害反弹/免疫
- 灵根共鸣(匹配+20%伤害)
- 五行相克(克制1.3/被克0.7)
- 元素抗性减伤(怪物对自身属性40%抗性)
- 被动触发效果(被打反烧/反毒/暴击反弹)
- 不灭金身(免死保留20%血)
- 战意沸腾(叠层机制)
- 道心通明(神通CD-1)

### 1.7 功法系统 ✅
- **39 个功法**: 6主修 + 14神通 + 19被动
- 品质: 凡品→灵品→玄品→地品→天品→仙品
- **主修**: 五行各一(替代普攻,带debuff)
- **神通**: CD自动释放,buff/debuff/护盾/控制
- **被动**: 数据驱动(15+种属性加成)
- 升级: Lv5上限,每级倍率+8%(主修/神通),+10%(被动)
- 按地图tier分级掉落(T1灵品→T7+仙品)

### 1.8 装备系统 ✅
- **7 槽位**: 兵器/法袍/法冠/步云靴/法宝/灵戒/灵佩
- **6 品质**: 凡器→太古神器
- **4 种武器类型**: 剑(均衡)/刀(爆发)/枪(持久)/扇(法术)
- **15 种副属性**: ATK/DEF/HP/SPD/会心率/会心伤害/破甲/命中/5种元素强化/灵气浓度/福缘
- **装备强化**: +10上限,+1~+5必成,+6起失败退1级(最低+5),+5/+10副属性突破
- **等级限制**: T1=Lv1, T5=Lv80, T8=Lv170, T10=Lv195
- 分类筛选,一键出售,悬浮属性提示

### 1.9 炼丹系统 ✅
- **灵草分级**: 7种(5五行+通用+仙灵草) × 6品质(凡品→仙品)
- **灵田地块**: 选种类种植→等待成熟→收获随机品质(灵田越高品质越好)
- **炼丹**: 选具体灵草品质→品质系数(1.0x~5.0x)→影响丹药效果和持续场次
- 9 个丹方(5战斗+4突破),失败只损失灵石不损失灵草
- 同种丹药buff覆盖

### 1.10 洞府系统 ✅
- **7 座建筑**: 聚灵阵/灵田/聚宝盆/演武堂/藏经阁/炼丹房/炼器房
- 前置解锁链,建造时间(6级+)
- 产出累积上限24h,一键领取
- 演武堂/藏经阁加成接入战斗

### 1.11 UI/UX ✅
- 水墨暗色主题
- 帮助文档按钮(完整游戏指南)
- 头像上传
- 装备/功法悬浮提示
- 战斗统计弹窗
- 掉落表弹窗

## 三、数据库表 (10 张)

| 表 | 用途 |
|----|------|
| users | 账号 |
| characters | 角色(含level/level_exp/avatar) |
| character_equipment | 装备(含base_slot/weapon_type/req_level/enhance_level) |
| character_skills | 已装备功法 |
| character_skill_inventory | 功法背包 |
| character_pills | 丹药背包(含quality_factor) |
| character_buffs | 激活buff |
| character_cave | 洞府建筑 |
| character_cave_plots | 灵田地块 |
| character_materials | 灵草背包(分品质) |

## 四、技术架构

```
前端: Vue3 + TypeScript + Vite + Pinia + Axios
后端: Express5 + TypeScript + MySQL8 + JWT
通信: RESTful API (35+ 接口)
战斗: 纯前端计算,结果上报后端
```
