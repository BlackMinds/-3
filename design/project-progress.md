## 一、已完成功能

### 1.2 角色创建 ✅
- 五行灵根选择（金/木/水/火/土），道号，头像上传，洗髓丹重置灵根

### 1.3 等级系统 ✅ **(v1.1 调整)**
- **上限 200 级**，打怪获得独立等级经验
- 经验曲线分段：
  - Lv1-30: `60 * lv^1.25`（原 80 * lv^1.3，-33%）
  - Lv31-80: `100 * lv^1.35`
  - Lv81-150: `180 * lv^1.42`
  - Lv151-199: `320 * lv^1.48`（原 350 * lv^1.5，Lv.200 现可达）
- 属性加成按段递增：
  - Lv1-50: 气血+5/攻击+2/防御+1/身法+1
  - Lv51-100: 气血+10/攻击+4/防御+2/身法+2
  - Lv101-150: 气血+20/攻击+8/防御+4/身法+3
  - Lv151-200: 气血+40/攻击+15/防御+8/身法+5

### 1.4 境界系统 ✅ **(v1.1 调整)**
- 8 大境界：练气 → 筑基 → 金丹 → 元婴 → 化神 → 渡劫 → 大乘 → 飞升
- 修为突破公式：
  - 练气期（tier=1）：`50 * stage^1.2`（线性快速突破）
  - 筑基+：`100 * exp_mul * (stage+1)^1.4`（原 1.6，平缓长线）
- 修为积满自动突破，解锁地图/功法/内容

### 1.5 闭关修炼系统 ✅ **(v1.1 调整)**
- 闭关 1-8 小时，消耗 `100 * realm_tier` 灵石/小时
- 修为产出：`80 * realm_tier * hours * (1 + stage * 0.1)`（原系数 50，+60% 提速）

### 1.6 地图历练系统 ✅ **(v1.2 调整)**
- **25 张地图**（T1~T10）
- T9: 太虚圣境/鸿蒙秘境/万界战场
- T10: 天道试炼/永恒之巅
- 每个地图 3-4 种普通怪 + 1 Boss
- **多怪波次**: 每次遭遇随机 1-5 只怪，逐个打，血量延续
- **T1-T3 地图灵石产出 +20%**（爽感期经济倾斜）
- **T7-T10 怪物 Power 平滑下调**（T7 ×0.70, T8 ×0.65, T9 ×0.60, T10 ×0.50），缓解飞升期数值断崖

### 1.7 战斗系统 ✅
- 服务端战斗引擎，最多 `50 * 怪物数` 回合
- **10 种 debuff**: 灼烧/中毒/流血/冻结/眩晕/减速/脆弱/攻击削弱/束缚/封印
- **8 种 buff**: 攻击提升/防御提升/速度提升/暴击提升/护盾/持续回血/伤害反弹/免疫
- 灵根共鸣（匹配+20% 伤害）
- 五行相克（克制 1.3/被克 0.7）
- 元素抗性减伤（怪物对自身属性 40% 抗性）
- 被动触发效果（被打反烧/反毒/暴击反弹）
- 不灭金身（免死保留 20% 血）
- 战意沸腾（叠层机制）
- 道心通明（神通 CD-1）

### 1.8 功法系统 ✅ **(v1.2 调整)**
- **46 个功法**: 6 主修 + 21 神通 + 19 被动
- 品质: 凡品 → 灵品 → 玄品 → 地品 → 天品 → 仙品
- **装备槽位按境界渐进解锁**:
  - 练气：1 主修 + 1 神通 + 1 被动（3 槽）
  - 筑基：1 主修 + 2 神通 + 2 被动（5 槽）
  - 金丹：1 主修 + 2 神通 + 3 被动（6 槽）
  - 元婴+：1 主修 + 3 神通 + 3 被动（7 槽）
- **升级倍率**: 每级 +15%（伤害/debuff 概率/buff 数值）
- **满级 Lv.5**，消耗残页升级（1/2/3/4 页/级）
- 按地图 tier 分级掉落（T1 灵品 → T7+ 仙品）

### 1.9 装备系统 ✅ **(v1.1 调整)**
- **7 槽位**: 兵器/法袍/法冠/步云靴/法宝/灵戒/灵佩
- **6 品质**: 凡器 → 太古神器
- **4 种武器类型**: 剑（均衡）/刀（爆发）/枪（持久）/扇（法术）
- **15 种副属性**: ATK/DEF/HP/SPD/会心率/会心伤害/破甲/命中/5 种元素强化/灵气浓度/福缘
- **装备强化**:
  - **+1~+6 必成**（原 +5 以下必成，+6 改为必成）
  - +7 75%、+8 55%、+9 40%、+10 25%
  - 失败退一级，最低保留 **+6**（原 +5）
  - +5/+10 副属性突破
- **等级限制**: T1=Lv1, T5=Lv80, T8=Lv170, T10=Lv195
- **品质权重前期上调**（T1: 白40/绿40/蓝17/紫3；T2: 白30/绿40/蓝22/紫7）

### 1.10 炼丹系统 ✅ **(v1.1 调整)**
- **灵草分级**: 7 种（5 五行 + 通用 + 仙灵草）× 6 品质（凡品 → 仙品）
- **品质系数**: 1.0x / 1.2x / 1.5x / 2.0x / 3.0x / 5.0x
- **灵田地块**: 选种类种植 → 等待成熟 → 收获随机品质（灵田越高品质越好）
- **种植时间缩短**（20/45/90/180/360/720 分钟，原为 30/60/120/240/480/960）
- **炼丹**: 选具体灵草品质 → 品质系数影响丹药效果和持续场次
- 9 个丹方（5 战斗 + 4 突破），失败只损失灵石不损失灵草
- 同种丹药 buff 覆盖

### 1.11 洞府系统 ✅ **(v1.1 调整)**
- **7 座建筑**: 聚灵阵/灵田/聚宝盆/演武堂/藏经阁/炼丹房/炼器房
- 前置解锁链，建造时间（6 级+）
- 产出累积上限 24h，一键领取
- **聚宝盆基础产出 800 灵石/h**（原 500，+60%）

### 1.12 宗门系统 ✅ **(v1.1 调整)**

- **加入门槛**: **Lv.15**（原 Lv.10，保证基础玩法熟悉）

- **每日捐献上限**: `100,000 × sectLevel` 灵石

- 日常任务（8 类）+ 周常任务（4 类）
- 宗门 Boss（5 只，按伤害排名分 4 档奖励）
- 宗门商店（17 道具，含强化保护符/大师符、装备鉴定符等）

- **强化保护符调整**: 成本 2000（原 3000）/周限 3（原 2）

## 五、技术架构

```
前端: Nuxt 3 + Vue 3 + TypeScript + Pinia + useCookie
后端: Nuxt Server Routes (Nitro) + PostgreSQL + JWT
通信: $fetch (93 个 API 端点)
战斗: 纯后端计算，防作弊
部署: Vercel + Neon (PostgreSQL) + GitHub Actions Cron
```

## 六、启动方式

```bash
# 前置：Node >= 22, PostgreSQL >= 14

# 1. 安装依赖
npm install

# 2. 初始化数据库
psql -U postgres -c "CREATE DATABASE xiantu_game;"
psql -U postgres -d xiantu_game -f server/database/migration.sql

# 3. 环境变量
cp .env.example .env  # 填入 DATABASE_URL / JWT_SECRET / CRON_SECRET

# 4. 开发服务器
npm run dev
# → http://localhost:3000
```
