## 六、Buff 系统 (8 种, 玩家自身)

| 类型 | 名称 | 效果 |
|------|------|------|
| atk_up | 攻击提升 | 当前攻击 × (1 + value) |
| def_up | 防御提升 | 当前防御 × (1 + value) |
| spd_up | 速度提升 | 当前速度 × (1 + value) |
| crit_up | 暴击提升 | 暴击率 + value |
| shield | 护盾 | 吸收攻击的伤害,值 = 主修攻击 × value |
| regen | 持续回血 | 每回合回 valuePercent × maxHp |
| reflect | 伤害反弹 | 受攻击时反弹 value 比例伤害 |
| immune | 免疫控制 | 期间免疫所有控制 debuff |

## 八、元素抗性

减伤公式: `伤害 × (1 - min(0.7, 目标对应抗性))`

- 怪物对**自身属性** 40% 抗性
- 怪物有 10% **控制抗性**
- 玩家抗性来自:灵根 / 被动 / 装备 / 丹药 buff,叠加上限 70%
