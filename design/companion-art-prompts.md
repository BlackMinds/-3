# 道侣立绘提示词清单（即梦专版）

> 2026-05-11 · 共 **60 张**立绘（性别 2 × 品质 6 × 性格 5）
> 工具：即梦 AI
> 命名：`<gender>_<quality>_<personality>.webp`（例：`female_5_冷艳.webp`）
> 放置：`public/img/companion/`

---

## 使用方法

每张图的 prompt = **【通用前缀】+【性别】+【性格描述】+【品质氛围】+【通用后缀】**

5 个模块拼装就能生成 60 张统一画风的图。先把这 5 段背熟，每张图改 2-3 个词就行。

---

## 模块 1：通用前缀（每张都用）

```
东方仙侠风格，中国古风插画，二次元立绘，单人半身像（头部到胸口），
精致五官，飘逸长发，灵气环绕，柔和电影光照，
```

## 模块 2：性别描述

| 性别 | 描述 |
|---|---|
| **female** | `一位绝美女子，` |
| **male** | `一位俊朗男子，` |

## 模块 3：性格描述（5 选 1，决定眉眼/服饰主色/氛围）

### 冷艳（蓝灰系）
```
银白色长发飘逸，淡紫色或冰蓝色眼眸，清冷绝美的神情，
身穿白色或冷蓝色长袍道袍，霜雪冰晶飘洒，
```

### 活泼（橙红系）
```
橙红色长发飞扬（女）/ 橙红短发飞扬（男），明亮金色眼眸，灿烂阳光的笑容，
身穿橙红色襦裙（女）/ 橙红色劲装（男），腰间斜挎长剑，火焰花瓣飘动，
```

### 温柔（粉紫系）
```
粉紫色长发披肩，柔和的浅蓝色眼眸，温婉如水的微笑，
身穿粉紫色长裙绣花，胸前莲花玉佩，樱花花瓣飘落，
```

### 高傲（金紫系）
```
深棕色长发盘高髻配金冠，凌厉的金色眼眸，冷艳挑眉，
身穿金色凤纹华贵长袍，胸前龙凤玉佩，凤凰虚影展翅，
```

### 俏皮（翠绿系）
```
碧绿色双马尾（女）/ 翠绿短发束起（男），狡黠灵动的翠绿眼眸，俏皮的笑容露齿微笑，
身穿青绿色短襦罗裙（女）/ 翠竹纹劲装（男），肩头停一只小狐狸或小麻雀，
```

| 品质 | 描述 |
|---|---|
| **0 凡品** | `朴素布衣线条简洁，无光晕，背景为山间小径或茅屋村落` |
| **1 下品** | `服饰简洁少量绣纹，淡绿色微光晕，灵气微现，背景为山林修行地` |
| **2 中品** | `服饰有银线刺绣花纹，中等蓝色灵气环绕，背景为云雾山峰` |

| **5 仙品** | `红色仙光冲天，云雾缭绕九霄，凤凰或真龙虚影盘旋，背景仙宫云海` |

## 即梦使用建议

1. **画面比例**：界面里选 `1:1` 方形
2. **画风模型**：选「**梦幻动漫**」或「**水彩绘画**」或「**国漫风**」之一并锁定，60 张全用同一个模型保证画风一致
3. **种子值（Seed）**：建议同一系列（同性别+同性格）锁定 seed，只调品质，这样仙品女冷艳和凡品女冷艳看起来像同一个人不同阶段
4. **批量出图技巧**：
   - 先出 5 张样图（建议先做仙品女冷艳 / 仙品男高傲 / 上品女温柔 / 中品男活泼 / 凡品女俏皮），观察画风是否统一
   - 风格稳定后批量出剩余 55 张
   - 每张可生成 4 张备选，挑最好的导出
