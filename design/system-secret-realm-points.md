# 秘境积分系统

> 用于换购强化石/附灵/突破丹的核心组队资源。本文档汇总 SR-1 ~ SR-11 全部 11 个秘境 × 3 难度的积分产出与消耗匹配。

## 一、积分发放公式

```
个人秘境积分 = floor( basePoints × 胜负系数 × 评分系数 × 个人分摊比 )
```

### 各因子取值

| 因子 | 取值 | 出处 |
|------|------|------|
| **basePoints** | 秘境难度配置基础积分（见下表） | `secretRealmData.ts` 各 difficulty.basePoints |
| **胜负系数** | 胜=1.0 / 败=0.3 | `team/start.post.ts` |
| **评分系数** | S=1.5 / A=1.25 / 其他=1.0 | `team/start.post.ts` |
| **个人分摊比** | 0.4 + 0.6 × 个人贡献度 | `team/start.post.ts:493` |

### 写入字段

- 累加到 `characters.realm_points`
- 失败保护期间（前 N 次）不扣次数，但积分按"失败 0.3 倍"发放
- "带人模式"（无次数队员）：积分 = 0，不扣次数、不记首通

- 跨秘境普通基线递增节奏：每升一档约 +33%，避免高境界产出爆炸

## 六、设计取舍

1. **节奏感保留**：跨秘境换图收益清晰（普通基线每档 +33%），但不会出现高境界爆炸性增长
2. **难度激励合理**：噩梦给普通 ×3，困难给普通 ×1.8 — 奖励挑战但不强迫必刷噩梦
3. **混元产出充足**：日产 32k-49k，能覆盖 1 周 ~25-35 万积分支出
4. **商店与掉落分工明确**：商店"定向购买"，秘境"主流路径"，互补而非替代
